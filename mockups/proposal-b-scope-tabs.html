Help & Reference
Scope Precedence (highest to lowest):
🔒 Local (.claude/settings.local.json) — personal project override, gitignored
📁 Project (.claude/settings.json) — shared with team via git
🏠 Global (~/.claude/settings.json) — your default for all projects
Profiles let you save provider+model configurations and quickly apply them to any scope. Use "Manage Profiles" to create, edit, and delete profiles.
Model Reset: When switching providers, models are reset to avoid accidental cross-provider model selection.
